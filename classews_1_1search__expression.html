Detailed Description

Base-class for all search expressions.

Search expressions are used to restrict the result set of a <FindItem/> operation.

E.g.

  • exists
  • excludes
  • is_equal_to
  • is_not_equal_to
  • is_greater_than
  • is_greater_than_or_equal_to
